🎬 한국어 영상 스크립트 추출기

이 프로그램은 MP4 동영상 파일에서 음성을 인식하여 타임스탬프가 포함된 스크립트(.txt)를 추출합니다.

✨ 주요 기능

  • Whisper 모델 선택 기능 (base, medium, large 등)
  • 실시간 작업 진행률 표시
  • 처리 완료된 스크립트 자동 저장

🚀 다운로드

사용하시는 운영체제에 맞는 버전을 다운로드하세요.

다운로드 (macOS .zip)

⚠️ 주의 사항
  • 프로그램을 처음 실행하면 음성 인식 모델(Whisper)을 다운로드하기 때문에 시간이 다소 걸릴 수 있습니다. (인터넷 연결 필수)
  • (1단계에서 imageio-ffmpeg가 실패했을 경우) 이 프로그램은 ffmpeg이 필요합니다. 만약 프로그램이 작동하지 않으면 여기에서 ffmpeg을 별도로 설치하고 시스템 PATH에 추가해야 할 수 있습니다.
